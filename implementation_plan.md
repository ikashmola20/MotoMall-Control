# نظام إدارة MotoMall — لوحة التحكم الشاملة

بناء نظام إدارة كامل ومتكامل لمتجر MotoMall للدراجات النارية والكهربائية، يعمل كمشروع مستقل في مجلد `Motomall opus 4.6 Control` باستخدام Next.js 16 + Tailwind CSS v4.

---

## تحليل الموقع الحالي

بعد دراسة الموقع بالكامل، هذا ما يحتويه:

| المكوّن | التفاصيل |
|---------|----------|
| **التقنية** | Next.js 16 + Tailwind CSS v4 + Firebase (Auth + Firestore) |
| **اللغة** | عربي RTL بخط Cairo + Inter |
| **المصادقة** | Firebase Auth (تسجيل/دخول/نسيان كلمة مرور) + Local Storage fallback |
| **البيانات** | Firestore مع mock data fallback (12 منتج، 4 أقسام، 10 براندات) |
| **الصفحات** | الرئيسية، الأقسام، المنتجات، البحث، الحساب، السلة، المفضلة |
| **الأنواع** | Product, Category, Brand, Order, UserAccount, Address |
| **الطلبات** | OrderStatus: processing → confirmed → in_transit → delivered |
| **الدفع** | الدفع عند الاستلام + تحويل مصرفي (مخطط: ZainCash, Qi, Visa) |

  - تمييز القيمة الأضعف بلون أحمر خفيف

مثال لمقارنة كهربائية:

```
┌────────────────────┬───────────────────┬───────────────────┐
│    المواصفة        │  NIU NQi GT       │  Super SOCO CPx   │
├────────────────────┼───────────────────┼───────────────────┤
│ 💰 السعر           │  2,850,000 IQD    │  2,200,000 IQD 🟢│
│ ⚡ قوة المحرك      │  3,000W    🟢     │  2,500W           │
│ 🔋 سعة البطارية    │  60Ah      🟢     │  45Ah             │
│ 📏 المدى           │  80 كم     🟢     │  65 كم            │
│ ⏱️ وقت الشحن      │  6 ساعات          │  4 ساعات   🟢     │
│ 🏎️ السرعة القصوى  │  70 كم/س   🟢     │  55 كم/س          │
│ ⚖️ الوزن           │  95 كجم           │  82 كجم    🟢     │
└────────────────────┴───────────────────┴───────────────────┘
```

> [!IMPORTANT]
> **قاعدة مهمة**: لا يمكن إضافة منتج إلى مقارنة إلا إذا كانت مواصفاته الفنية مُدخلة بالكامل. النظام سيُظهر تنبيه عند محاولة إضافة منتج بدون مواصفات.
